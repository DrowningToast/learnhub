# Learnhub

# วิธีการติดตั้งสำหรับ Production
วิธีการติดตั้งตัวโครงงานนั้นมีด้วยกัน 2 วิธีได้แก่
1. การติดตั้งเป็น Docker Container ด้วย Docker Compose
2. การติดตั้งเป็น Virtual Machine ด้วยไฟล์ OVA

# วิธีการติดตั้งที่ 1 ด้วยการใช้ Docker
ในการติดตั้งประเภทนี้นั้น จำเป็นที่จะต้องสร้าง Docker Image ของตัวเว็บไซต์โดยที่มีขั้นตอนดังต่อไปนี้

# สิ่งที่จำเป็นต่อการติดตั้ง

- node.js เวอร์ชั่น 20 เป็นขั้นต่ำ
- Docker ที่กำลังรันอยู่

# ก่อนทำการติดตั้ง กรุณาตรวจสอบว่าไม่มี container หรือ process ใดๆ กำลังใช้งาน port 80, 5173, และ 5432

1. เริ่มต้นด้วยการรัน `npm run compose:prod` บน directory ของตัวโปรเจค โดยขั้นตอนนี้อาจจะใช้เวลาถึง 3 - 20 นาทีแล้วแต่ความเร็วของอินเตอร์เน็ตและความเร็วในการประมวลผลของเครื่อง

สามารถตรวจสอบสถานะการทำงานได้ว่าสำเร็จหรือไม่ โดยหากสำเร็จจะแสดง output บน terminal คล้ายกับรูปดังต่อไปนี้ https://drive.google.com/file/d/1X-qO66voEd98Ok5sFZI9LCjj2sBvxIjN/view?usp=sharing

2. เมื่อทำการรันเสร็จสิ้น ต่อไปจะเป็นการทำ migrations และ seed ข้อมูลเข้าไปในตัวเว็บไซต์โดยให้ทำการรัน `npm run migrate:prod` ขั้นตอนดังกล่าวอาจใช้เวลา 1 - 5 นาทีแล้วแต่ความเร็วของการประมวลผลของคอมพิวเตอร์

สามารถตรวจได้ว่าขั้นตอนที่ 2 นั้นทำงานสำเร็จหรือไม่ จากการที่ process การ seeding นั้นเสร็จสิ้น

3. ทำการตรวจอบสถานะของเว็บไซต์โดยการเข้าไปที่ http://localhost เพื่อเข้าเยี่ยมชมเว็บไซต์
